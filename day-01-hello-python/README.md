> **Today in one line:** Write and run your very first Python program using the `print()` function.
## Why this matters

Programming is about giving clear instructions to a computer to solve problems. Before building websites, games, AI models, or automation tools, you must first learn how Python understands and executes simple instructions.

Today you'll write your first Python program and learn how the `print()` function displays information on the screen. This small step is the foundation for everything you'll build in the coming days.
### What does `print()` do?

The `print()` function displays information on the screen. It is one of the first functions every Python programmer learns because it helps us see the output of our programs.

Example:

```python
print("Welcome to Python!")
```

Output:

```text
Welcome to Python!
```

Notice that the text is written inside quotation marks. Python treats anything inside quotation marks as text and displays it exactly as written.
"What does print() do?"
### Common Mistake of the Day

A common beginner mistake is forgetting quotation marks around text.

Incorrect:

```python
print(Hello)
```

Error:

```text
NameError: name 'Hello' is not defined
```

Correct:

```python
print("Hello")
```

Python treats text inside quotation marks as a string. Without quotation marks, Python looks for a variable named `Hello`.
